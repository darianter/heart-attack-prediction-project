# heart-attack-prediction-project
## Файлы
- heart_project.ipynb -- ноутбук с исследованием, обучением моделей и выводами
- requirements.txt -- требуемые модули
- main.py -- приложение на FastApi
- test_request.ipynb -- ноутбук для тестирования приложения
- predictions.csv -- предсказания, полученные на тестовой выборке (id -- id пациента, prediction -- предсказание) 

## Инструкция по работе с приложением
1. Получить файл модели (heart_model.pkl), запустив ноутбук с обучением модели (heart_project.ipynb)
2. Загрузить требуемые модули из requirements.txt
3. Запустить приложение (main.py)
4. Запустить ноутбук для тестирования приложения (test_request.ipynb)
